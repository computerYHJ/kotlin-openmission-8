# Wokrout Project
간단한 운동기록/사용자 관리 어플 입니다.

이전에 개발해봤던 어플을 기반으로 언어를 Kotlin으로 변경하였습니다

---

## 주요 화면 구성

### 1. 초기화면
- 로그인 화면 이동
- 회원가입 화면 이동
- 아이디 찾기 화면 이동

### 2. 로그인 화면

- 아이디를 입력하는 EditText
- 비밀번호를 입력하면 EditText
- 아이디 찾기 화면 이동
- 비밀번호 찾기 화면 이동

### 3. 회원가입 화면

- 회원가입 할 아이디를 입력하는 EditText
- 아이디를 중복체크하는 버튼
- 회원가입 할 비밀번호를 입력하는 EditText
- 입력한 비밀번호를 다시 입력하는 EditText
- 이름을 입력하는 EditText
- 회원가입 할 이메일을 입력하는 EditText
- 이메일을 중복체크 하는 버튼
- 월별 목표 운동횟수를 저장하는 EditText(미구현)
- 회원가입 버튼

### 4. 아이디 찾기 화면

- 찾고싶은 계정의 이름을 입력하는 EditText
- 찾고싶은 계정의 이메일을 입력하는 EditText
- 입력한 정보를 가지고 아이디를 찾는 버튼
- 아이디를 보여주는 TextView(평시엔 숨겨져 있음)

### 5. 비밀번호 찾기 화면

- 찾고싶은 계정의 아이디를 입력하는 EditText
- 찾고싶은 계정의 이메일을 입력하는 EditText
- 입력한 정보를 가지고 비밀번호를 찾는 버튼

### 6. 메인 화면(로그인 성공 시)

- 사용자 이름과 목표 횟수를 기반으로 환영 메시지를 표시
- 진행률(이용률)을 원형 Progress UI로 시각화
- 사용자가 등록한 이용권의 종료 날짜 표시
- 이용권 등록(수정) 버튼을 통해 이용권 등록(수정) 화면 이동 
- 이용시간 확인 버튼으로 사용자의 이용 가능 시간 다이얼로그 출력
- 사용자의 누적 운동 횟수와 남은 횟수 안내
- 내역 확인하기 버튼으로 운동 기록 상세 페이지로 이동(미구현)
- 오늘의 운동 누적 시간 표시
- 운동 시작하기 버튼을 누르면 타이머 작동, 다시 누르면 누적운동횟수 + 1

### 7. 이용권 등록(수정) 화면

- 이용권이 시작되는 날짜를 보여주는 TextView
- 이용권이 시작되는 날짜를 선택하는 버튼
- 이용권이 종료되는 날짜를 보여주는 TextView
- 날짜 선택 버튼 클릭 시, 연도,월,일 선택하는 다이얼로그 출력
- 헬스장 시작 시간을 표시하는 TextView
- 헬스장 종료 시간을 표시하는 TextView
- 시간 선택 버튼 클릭 시, 시간을 선택하는 다이얼로그 출력

## 프로젝트 구조

```
/app
 ├── activity/               # 각 화면(Activity)
 ├── viewModel/              # 화면 상태 관리 ViewModel
 ├── model/                  # 데이터 클래스(User, State 등)
 ├── repository/             # Firebase 데이터 처리
 ├── validator/              # 입력값 검증 로직
 └── res/                    # XML 레이아웃, drawable, values
```

---

## 기술 스택
- Language: Kotlin
- Architecture: MVVM
- Jetpack Components: ViewModel, StateFlow/EventFlow, ViewBinding
- Firebase: Authentication, Firestore
